## Technical Specifications

- **Format:** SVG (Scalable Vector Graphics)
- **Dimensions:** 500×500px (scales infinitely)
- **Color Space:** RGB
- **Transparency:** None (solid backgrounds)
- **File Size Range:** ~350 bytes - ~3KB

## Usage Guidelines

### Do's ✅

- Use logos at their original aspect ratio
- Ensure adequate contrast with surrounding elements
- Export at appropriate resolution for your use case
- Test logos against different backgrounds when possible
- Use gradient logos for premium/feature presentations

### Don'ts ❌

- Don't stretch or distort logos
- Don't change the gradient colors
- Don't add drop shadows or effects
- Don't place logos on busy backgrounds
- Don't use low-contrast combinations (e.g., white logo on light gray)

## Implementation Examples

### Next.js / React

```jsx
import Image from 'next/image';

// Full logo on dark background
<Image
  src="/logos/logo_1.svg"
  alt="10XRX Logo"
  width={500}
  height={500}
/>

// Icon only for favicon
<link rel="icon" href="/logos/logo_9.svg" type="image/svg+xml" />
```

### HTML

```html
<!-- Full logo -->
<img src="/logos/logo_1.svg" alt="10XRX Logo" width="200" height="200" />

<!-- Responsive logo -->
<picture>
  <source srcset="/logos/logo_1.svg" media="(prefers-color-scheme: dark)" />
  <source srcset="/logos/logo_4.svg" media="(prefers-color-scheme: light)" />
  <img src="/logos/logo_4.svg" alt="10XRX Logo" />
</picture>
```

### CSS Background

