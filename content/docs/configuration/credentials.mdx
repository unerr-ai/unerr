Login is optional. unerr works fully offline. You only log in to use the dashboard's
cross-machine view and team features.

<Callout type="info">
Offline by default: if you never log in, nothing leaves your machine. Indexing, the graph,
notes, and tools all run locally.
</Callout>

## When you log in

The process manager (`unerrd`) reports a small inventory plus a heartbeat to your account,
so the dashboard can show all your machines in one place. It reports:

| Reported | Example |
|---|---|
| Repo paths | `/Users/you/project` |
| Git origin | the repo's remote URL |
| Ports | the dashboard / process ports in use |
| Process status | running / stopped |
| Heartbeat | a periodic "still alive" ping |

Your code is not sent.
