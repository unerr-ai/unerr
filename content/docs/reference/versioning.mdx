unerr uses [semantic versioning](https://semver.org/) (`MAJOR.MINOR.PATCH`).

- Current release: **0.3.0**
- Ships as a self-contained binary; also published to npm as `@unerr-ai/unerr`.
- See [Installation](/docs/getting-started/installation) for all install channels.

## Check your version

```bash
unerr --version
```

## Upgrade

| Install channel | Upgrade command |
|---|---|
| curl \| bash | Re-run the install command (it fetches the latest) |
| Homebrew | `brew upgrade unerr` |
| Scoop | `scoop update unerr` |
| npm | `npm install -g @unerr-ai/unerr@latest` |
