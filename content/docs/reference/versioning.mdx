unerr uses [semantic versioning](https://semver.org/) (`MAJOR.MINOR.PATCH`).

- Current release: **0.3.0**
- npm package: `@unerr-ai/unerr`

## Check your version

```bash
unerr --version
```

## Upgrade

```bash
npm install -g @unerr-ai/unerr@latest
```

Use the matching command for your global package manager (`pnpm add -g` or `yarn global add`) if you didn't install with npm.
