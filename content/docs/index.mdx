unerr is local-first code intelligence for AI coding agents. It runs entirely on your machine, needs no account, and works offline.

It indexes your repo into a local code graph and serves graph-backed tools and rules to the agent as it works. The agent gets the live call graph plus the rules tied to each entity at the moment it edits — and those rules re-anchor when the code moves, so they don't go stale.
