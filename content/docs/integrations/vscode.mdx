---
title: VS Code
description: Set up unerr for VS Code, including the MCP config and instruction file.
---

VS Code gets unerr's MCP tools and an instruction file. It does not get skills
or hooks (those are Claude Code only).

## Setup

1. Install the CLI (once per machine):

   ```bash
   curl -fsSL https://raw.githubusercontent.com/unerr-ai/unerr/main/install | bash
   ```

   On Windows, or to use Homebrew, Scoop, or npm, see [Installation](/docs/getting-started/installation).

2. Go to your repo and install for VS Code:

   ```bash
   cd /path/to/your/repo
   unerr install vscode
   ```

## What it writes

| Path | What it is |
|---|---|
| VS Code MCP config | The MCP config VS Code reads. Shape: `{ "command": "<absolute path to unerr>", "args": ["--mcp"] }`. |
| `.github/copilot-instructions.md` | Instruction file telling the agent to prefer unerr's tools. |

## Confirm the tools appear

1. Reload the VS Code window so it picks up the MCP config.
2. Check VS Code's MCP server list and confirm the unerr server is connected.
3. The unerr tools (`search_code`, `get_references`, `file_read`,
   `file_outline`, `unerr_context`, `fetch_url`, `unerr_track`) are now available
   to the agent.

You can also check unerr's side:

```bash
unerr status
```
