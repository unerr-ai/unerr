unerr works with 16 agents. Run `unerr install <agent>` in a repo using the id below. The command writes the MCP config and, where the agent supports one, a project instruction file telling the agent to prefer unerr's tools.

## Claude Code gets the most

Claude Code receives skills, hooks, read-routing, and the edit gate on top of the MCP tools. Every other agent gets the MCP tools and, where supported, an instruction file that tells it to prefer unerr's tools.
